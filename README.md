# CodeReviewer

Projeto em C# para identificação de imagens utilizando o modelo multimodal LLaVA.
O objetivo é explorar como modelos de linguagem multimodal podem ser integrados em aplicações .NET para reconhecer e descrever imagens.

## Funcionalidades
Carregamento de imagens para análise.

Integração com modelo LLaVA para identificação e descrição.
Estrutura modular em .NET, facilitando expansão e manutenção.
Projeto pronto para ser compilado e executado em ambiente desktop.

## Como Executar

#### 1. Clone este repositório:
```
git clone https://github.com/sorza/Image_Identifier.git
```

#### 2. Abra a solução no Visual Studio ou VS Code.

#### 3. Compile o projeto:
```
dotnet build
```

#### 4. Execute a aplicação:
```
dotnet run
```
